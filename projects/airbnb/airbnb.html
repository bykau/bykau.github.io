Predict Next Destination for Airbnb Users

Dataset

Airbnb user dataset (1.1GB)

Technologies & Algorithms

python, pandas, matplotlib, xgboost, sklearn, ipython notebook, cross validation, grid search

Project Description

In this project I aim to predict in which country a new Airbnb user will make his or her first booking. The dataset consists of a list of users along with their demographics, web session records, and some summary statistics. I did an extensive feature engineering based on the user session data as well as I tuned the xgboost model using sklearn Randomized Search CV.

Results

The solution I proposed got into the top 10% at the Airbnb New User Bookings Kaggle Competition.
